# Universal Calculator 隐私权政策

七九在线科技（以下简称“我们”或“79team”）非常重视用户隐私，并致力于保护您的个人数据。本隐私权政策旨在向您说明 Universal Calculator（万能计算器）浏览器扩展程序（以下简称“本扩展”）如何收集、使用和保护您的数据，以确保本扩展符合 Chrome Web Store 开发者政策要求。

---

## 一、我们收集与处理的数据

根据本扩展的功能设计，我们严格遵循**最小化数据收集原则**，绝大多数计算与解析完全在本地运行：

### 1. 本地存储数据 (chrome.storage.local / localStorage)
- **偏好设置与状态记录**：我们会将您的使用首选项（如主题样式 `theme_mode`、语言设置 `user_lang`、计算器简化/科学版本切换状态等）存储在您的本地浏览器中。
- **本地存储目的**：仅用于保持用户界面的连贯体验，无需每次重新配置。

### 2. 汇率数据接口请求 (host_permissions)
- **实时汇率更新**：为提供最新、准确的汇率换算数据，本扩展在您切换至汇率模块时会向公开的汇率 API 发起安全网络请求。
- **请求内容**：仅请求基准货币的最新报价，不上传、不包含任何可识别个人身份的信息或历史操作。

### 3. AI 助手对话与深度解析 (host_permissions)
- **AI 深度解析**：当您使用内嵌 AI 解析功能（如 BMI 健康解析、房贷还款建议）时，本扩展会直接请求您自定义配置的 AI 服务提供商（如 OpenAI、Google Gemini、Anthropic 等）。
- **用途**：仅用于将当前计算结果（如身高、体重、BMI 指数或贷款本息）传递给大模型以获取专业解读，大模型返回后数据立即在界面呈现，扩展自身绝不在任何云端服务器保留您的计算与对话历史。

---

## 二、数据用途与处理规则

1. **完全本地计算**：科学计算、利息房贷推演、物理单位换算、财务大写转换等核心计算服务全部在您的本地浏览器沙箱中安全运行，**绝不会上传到我们的任何服务器**。
2. **AI API 密钥安全**：您在设置中填写的 API 密钥完全保存在您的本地浏览器中。发起 AI 对话时以直连方式与对应大模型 API 通信，**我们不中转、不截获也不保存您的 API 密钥与任何提示词内容**。
3. **无追踪与无广告**：本扩展不包含任何广告追踪插件，不搜集，亦不使用任何用户行为分析工具。

---

## 三、数据的共享与转让 (Data Sharing and Transfer)

我们郑重承诺：
1. **绝不向任何第三方出售、转让、共享或披露您的任何数据**。
2. **所有数据不会被用于除本扩展功能之外的任何商业盈利目的**（如广告推送、个性化画像建立等）。
3. 所有本地存储数据受浏览器沙箱安全框架保护，未获授权的第三方网站或程序无法读取。

---

## 四、第三方服务的隐私政策

当您使用 AI 解析功能时，传输的计算参数将受您配置的第三方大模型服务商（例如 OpenAI, Google 等）隐私权政策约束。我们建议您在使用前了解其对应的隐私声明。

---

我们可能会根据功能演进或 Chrome Web Store 的政策规范适时更新本隐私权政策。如有更新，我们会及时在本页面和发布仓库中予以公告。
